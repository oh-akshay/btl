Icons for categories
====================

Place optional SVG/PNG icons here. The app will try to load an icon for each category:

- assets/icons/language.svg
- assets/icons/art.svg
- assets/icons/music.svg
- assets/icons/stem.svg
- assets/icons/fitness.svg
- assets/icons/public-speaking.svg
- assets/icons/chess.svg
- assets/icons/robotics.svg

If an icon is missing, the UI falls back to an emoji.
